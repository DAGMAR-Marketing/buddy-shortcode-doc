Buddy shortcode doc for pomi

Shortcode

[buddy]

The shortcode will work on its own without parameters, but will default to using the season interruption form with no attributionId option value included. The config file's subdomain ("embed" or "staging.embed"), the stage option, and the partnerID option will all toggle based on whether the offer form is output on the production domain (getpomi.com) or a non-production domain.

Attributes (Parameters)

To use a shortcode attribute, use the following syntax: [buddy attribute="value"].

Example: [buddy form="pp"].

You may use any number of attributes within a single shortcode, using the same syntax as above.

Example: [buddy form="pp" stage-stg="TESTING" aid="partner-xyz"].

  • form
    • Currently, form="pp" and form="si" are the only accepted values. form="pp" will output the accident coverage form ("player protect") and the form="si" will output the season interruption form.
    • No form attribute, or an attribute value that is not recognized, will default to the season interruption form.
    • The attributionId is not included in the output of these forms. A separate shortcode attribute is used to output this option (see below).
  • stage-prd
    • Use this attribute to override the default stage option value for the production domain (getpomi.com)
    • The default value is PRODUCTION
  • stage-stg
    • Use this attribute to overridee the default stage option value for all non-production domains
    • The default value is STAGING
  • config-path
    • Use this attribute to override the default path for the config file.
    • Default value for the accident form is pomi/gaig-pomi-accident-config
    • Default value for the season interruption form is si/gaig-gaig-si-config
  • pid-prd
    • Use this attribute to override the default partnerID option on the production domain (getpomi.com).
    • Default value for both forms is p-19g6jlnnerv7e
  • pid-stg
    • Use this attribute to override the default partnerID option on non-production domains.
    • Default value for the accident form with no attributionId value set is p-test-f7vrennlj
    • Default value for the accident form with an attributionId set is p-buddytest
    • Default value for the season interruption form is p-buddytest
  • ion
    • Use this attribute to override the default ion option value
    • Default value for the accident form is GAIG_POMI_ACCIDENT
    • Default value for the season interruption form is GAIG_GAIG_SI
  • aid
    • Use this attribute to add an attributionId option value
    • Neither form includes an attributionId option value by default.
  • theme
    • Use this attribute to override the default theme
    • Default value for the accident form is themeBase (i.e. options.theme = config.themeBase)
    • Default value for the season interruption form is themePomi (i.e. options.theme = config.themePomi)
  • events
    • Use this attribute to override the default event set
    • Default value for the accident form is userEvents (i.e. options.onUserEvent = config.userEvents)
    • Default value for the season interruption form is pomiUserEvents (i.e. options.onUserEvent = config.pomiUserEvents)
